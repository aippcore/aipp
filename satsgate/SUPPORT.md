# Support

## Where to ask questions

- For bugs and feature requests: open a GitHub Issue
  - https://github.com/Mike-io-hash/satsgate/issues

- For security issues: see `SECURITY.md`

## What to include

To help us reproduce issues quickly, please include:
- your satsgate base URL
- the endpoint you called
- the HTTP status code
- the response body (remove secrets)
- your deployment mode (hosted vs self-hosted)
